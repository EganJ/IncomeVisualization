About the Visual

This visual is meant to convey information about income distributions in any city or zip code.

Reading the Visual

Once you search for a region, this text will be replaced with 100 scaled images, each representing the household income that each percentile of the population would earn. Images are scaled so that a 100px image represents the average American household income. Images are scaled proportionaly, to a household making twice that would be twice as large. To get exact values, hover over an image once you search.

Searching

To search, just enter a 5-digit zip code or enter your city and state.

Ex: 12345 or Somecity, Ca. Don't forget the comma!

For most accurate results, please use zip codes.This is because data is currently gathered for only one zip code at a time, and cities can have multiple zip codes, and one is chosen more or less at random.
